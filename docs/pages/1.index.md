This module is a wrapper around [Supabase](https://supabase.com/) for [Nuxt](https://v3.nuxtjs.org).

## Features

<list>

- Nuxt 3+ ready
- Vue 3 composables
- Authentication support
- Use supabase-js isomorphic client
- TypeScript support

</list>
